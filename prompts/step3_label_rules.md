# Step 3 · 命名规则（v4）

用于 Step 3 各阶段（层内合并、纵向装配、新建父节点）的命名参考。

1. 结构
   - `动作 + 对象` 或 `策略 + 场景`，示例：“冷链奖补”“算力调度支持”“融资贴息”。
   - 控制在 6-12 个汉字，避免冗余词（“推进”“相关工作”等）。
2. 语义
   - 同层保持粒度一致：L4 用“具体措施”，L3/L2 用“主题+工具”。
   - 不含地名、年份、百分比、金额等不稳定信息。
3. 禁忌
   - 避免“及以上 / 以下”“其他”“综合”等模糊词。
   - 不直接引用法规编号。

示例：
- ✅ “冷链运营奖补”“制造业技改贴息”“算力枢纽建设支持”
- ❌ “关于推进冷链产业高质量发展的若干措施”（过长且含虚词）
