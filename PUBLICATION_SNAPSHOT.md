# Local Publication Snapshot

The public repository is derived from the local paper-version directory:

`<LOCAL_SOURCE_ROOT>/roundC_v4`

During packaging, an exact local-only archive was created at:

`_local_archive/roundC_v4_paper_snapshot_2026-05-17.zip`

A SHA256 manifest for the archived source tree was created at:

`_local_archive/MANIFEST.sha256`

The `_local_archive/` directory is intentionally ignored by Git because it preserves the author's private full snapshot, including files that should not be published directly, such as local environment files.
