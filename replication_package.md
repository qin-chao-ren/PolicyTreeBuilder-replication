# Replication Package Description

This file describes the planned contents of the replication package.

## 1. Processed Policy-Action Dataset

The processed dataset will contain the extracted and normalized policy-action nodes from the collected Chinese air-cargo policy documents.

## 2. Prompts

The prompt files will include the instructions used for:

- policy-action extraction;
- semantic granularity calibration;
- L1 thematic anchoring;
- hierarchical tree construction;
- evaluation.

## 3. Analysis Scripts

The scripts will support data processing, node organization, descriptive statistics, and visualization.

## 4. Intermediate Outputs

Intermediate outputs will be provided to support transparency and reproducibility.

## 5. Final Policy-Action Tree

The final hierarchical policy-action tree will be released in both machine-readable and human-readable formats.

## Current Status

The replication package is under preparation and will be made publicly available after final verification.
