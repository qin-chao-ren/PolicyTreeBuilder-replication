# PolicyTreeBuilder Replication Package

This repository provides the replication package for the paper:

**PolicyTreeBuilder: An LLM-assisted framework for constructing hierarchical policy-action trees from Chinese air-cargo policy documents**

## Repository Status

This repository is currently being prepared for public release. The complete replication package will include:

- processed policy-action dataset;
- prompts used for policy-action extraction, semantic calibration, and tree construction;
- analysis scripts;
- intermediate outputs;
- final policy-action tree.

The complete materials will be added after final checking and anonymization.
